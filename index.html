01 — ABOUT

What is "INNOVE"?

福島イノベとは？

「福島イノベーション・コースト構想」は、東日本大震災及び原子力災害によって失われた浜通り地域等の産業を回復するため、新たな産業基盤の構築を目指す国家プロジェクト。ロボット・ドローン、エネルギー、農林水産業など、世界の最先端に挑む現場が、福島・浜通りに集まっています。

本プログラムは、公益財団法人 福島イノベーション・コースト構想推進機構の事業として実施されます。

02 — PROGRAM

Two Challenges

選べる、または連動する、２つのチャレンジ。

01

MEDIA PROGRAM

広報コンテンツ制作プログラム

02

SOLUTION PROGRAM

課題解決力育成プログラム

地域のリアルな課題に、高校生の視点で挑む。

07 — MENTOR

With Mentors

プロと一緒に、考えて、つくる。

地域や事業づくりに詳しいメンター、動画・デザイン・広報の専門スタッフが、高校生の活動をサポートします。「何を伝えるか」から、「どう見せるか」まで、チームで一緒に考えていきます。

LOCAL GUIDE

地域伴走メンター

浜通りで活動する起業家・地域プレイヤーが、課題発見から提案まで伴走します。

CREATION

制作メンター

映像・写真・デザインのプロクリエイターが、制作の基礎から表現まで指導します。

PUBLICITY

広報メンター

広告・SNS運用の専門スタッフが、「届く発信」のつくり方をサポートします。

SOLUTION

課題解決メンター

課題整理や企画立案の専門家が、アイデアを「提案」に磨き上げる過程をサポートします。
